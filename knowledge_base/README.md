# 知识库 - AI 桌面助手

此文件夹中的文件会被 AI 助手读取并作为上下文参考，帮助 AI 更准确地回答与这些资料相关的问题。

## 支持的文件类型

| 类型 | 格式 |
|------|------|
| 纯文本 | `.txt` |
| Markdown | `.md` |
| 代码 | `.py`, `.js`, `.ts`, `.jsx`, `.tsx`, `.java`, `.c`, `.cpp`, `.h`, `.hpp` |
| 配置 | `.json`, `.yaml`, `.yml`, `.xml`, `.toml`, `.ini`, `.cfg`, `.conf` |
| 网页 | `.html`, `.css`, `.scss`, `.less` |
| 数据 | `.csv` |
| 脚本 | `.sh`, `.bat`, `.ps1` |
| 文档 | `.rst` |

## 文件限制

- **单文件最大**: 512 KB（超出将被跳过）
- **编码**: 仅支持 UTF-8 编码的文本文件
- **不支持**: 二进制文件、图片、PDF、Word 文档、压缩包等非纯文本格式
- **总数上限**: 最多 50 个文件（超出后多余文件被忽略）

## 目录结构

支持子目录嵌套，AI 会递归读取所有支持的文件。

```
knowledge_base/
├── README.md          # 本说明文件（不会作为知识库内容读取）
├── project_rules.md   # 项目规范示例
├── api_docs/
│   └── endpoints.txt  # API 文档示例
└── notes/
    └── reference.txt  # 参考资料示例
```

## 使用方式

1. 将需要的参考文件放入此文件夹
2. 重启 AI 桌面助手或在设置中开启「启用知识库」
3. AI 将自动读取文件内容并在对话中作为上下文参考

## 注意事项

- 知识库内容会在每次对话中作为系统提示词的一部分发送给 AI，建议总内容不超过 10,000 字
- 文件过多或过大可能影响响应速度
- 敏感信息请勿放入此文件夹
- 修改文件后需重启应用或切换模型以重新加载
